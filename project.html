Sekilas Tentang Proyek yang Pernah Saya Buat

Traseon Farmer

Website Traseon Farmer ini saya buat menggunakan framework CodeIgniter 3. Website ini digunakan untuk mengelola data, manajemen supply chain dan mentracking data-data yang berhubungan dengan proses bisnis milik Yayaysan Hutan Tropis (YHT).

GLabs

Website GLabs ini saya buat menggunakan framework Laravel. Website ini memiliki fungsi untuk mengelola data peminjaman alat labarotarium yang ada di Laboratorium Kimia UII Yogyakarta. Terdapat banyak user meliputi dosen, mahasiswa, dan para peneliti yang menggunakan website tersebut.
